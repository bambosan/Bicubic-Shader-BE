BSBE Lite

A Shader/Renderdragon Materials for Minecraft: Bedrock Edition that aims to be beautiful look with colorful tones and some features like underwater caustic, 2D cloud, water reflection, water bump, etc.






Changelog


  • minor improvements and optimizations
  • added Minecraft vanilla AO
  • added support for MCBE's Windows and iOS versions
  • now this shader only supports MCBE with the Renderdragon engine
