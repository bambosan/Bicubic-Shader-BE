Bicubic Shaders BE

About

This is shaders (Renderdragon materials) for Minecraft: Bedrock Edition that aims to be beautiful look with colorful tones and some visual features

Feature

  • Beautiful sky color
  • Beautiful fluffy cloud (fake 3d cloud)
  • Sky and cloud reflection on water
  • Water bump
  • Underwater caustic
  • Colored ambient lighting with tonemapping
  • Colored block light source
  • Fake diffuse lighting
  • Plants and foliage waing (curently on old version)
  • Water waving (curently on old version)
  • and more..

Changelog v 3.0 to 3.1

  • Added support for MCBE with Renderdragon engine
  • Added foggy effect
  • Added Minecraft vanilla AO
  • New tonemapping
  • Improve cloud
  • Improve sky
  • Improve water reflection
  • More vibrant color
  • Change underwater fog
  • Remove sun reflection on water
  • Some optimizations

Warning

  • From version 3.1 only works for Minecraft: Bedrock Edition with Renderdragon engine
  • Install the resoucepack BSBE RD Material support for activate some features especially cloud and water detection
